[−][src]Module greenhorn::pipe

This modules defines an interface for the communication interface between frontend and backend.

The central trait is the Pipe type, which exposes both a Sender and a Receiver.

A Sender can be used to send TxMsg message, i.e. message from backend to the frontend. A Receiver can be used to receive RxMsg message, i.e. message from frontend to the backend.
